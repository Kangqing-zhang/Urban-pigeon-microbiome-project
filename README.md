# Urban pigeon microbiome

Analysis materials for the published paper:

**Seasonal Dynamics of the Gut Microbiome in Urban Feral Pigeons Are Associated With Environmental Conditions, Not With Diet Shifts**

Ecology and Evolution, 2026  
DOI: https://doi.org/10.1002/ece3.73682

## Authors

Kangqing Zhang, Henri Zomer, Gerrit Potkamp, Joana Falcao Salles, B. Irene Tieleman, Maurine W. Dietz

## Repository description

This repository contains analysis scripts, metadata, and selected data files related to the study of seasonal variation in the gut microbiome of urban feral pigeons.

The repository is organized by analysis type, including diet-related analyses, microbiome analyses, environmental association analyses, Procrustes analysis, and partial least squares path modelling.

## Project structure

- `Diet/`: diet-related data and analysis scripts
- `Microbiome/`: microbiome data and analysis scripts
- `dbRDA/`: environmental association analyses
- `Procrustes/`: Procrustes analysis
- `PLSPM/`: partial least squares path modelling
- `README.md`: overview of the repository

## Data availability

The datasets supporting this article are available in DataverseNL (https://doi.org/10.34894/9UNDGT). These include the raw sequences, metadata, and R scripts used in this study.
